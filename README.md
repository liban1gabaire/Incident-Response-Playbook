# Incident Response Playbook — Project 3

## Overview
Developed a structured incident response playbook for brute force attacks, 
based on real detection and investigation work performed in Project 1 
(SOC Home Lab). Covers the full incident lifecycle from detection through 
lessons learned.

## Skills Demonstrated
- Incident response methodology (NIST-aligned lifecycle)
- Threat containment and eradication planning
- Security documentation and process design
- Practical application of SIEM alert data to response procedures
